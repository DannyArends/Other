<h2>Sourcecode</h2>
"In computer science, source code is any collection of statements or declarations written in some human-readable computer programming language. Source code allows the programmer to communicate with the computer using a reserved number of instructions. " - <a href="http://www.wikipedia.org">wikipedia</a>

With this in mind the aim of biosource.nl is to provide biologists & researchers with tailormade software packages to help answer their questions.
